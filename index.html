CSS is an acronym for Cascading Style Sheet, which means flowchart style pages, and CSS technology is used in designing web pages so that the entire site can be controlled in terms of font type, color, or size without having to write those codes or repeat them in Each page within the site, and therefore it is possible through only one file to make any adjustments to all files, which saves a lot of effort and time. And the function of this technique is to control how the webpage is displayed without interfering with the content, which facilitates the process of managing the site. Modifying one CSS file will affect the design of all pages of the site. CSS files simplify the process of designing and managing websites, and solve many problems that any website developer might encounter. For example, if you designed your site using HTML, assuming that this site consists of five pages, then you want after a while. Development and modification of the way this website is presented ... You need here to redesign the five pages again, this is very easy and you have to start now. Assuming that your site contains more pages, so be 50 or 100 pages, then surely you will Feel free before proceeding with the development and modification ... As for if you have designed your site from the beginning using CSS technology, all you have to do is Just one file. CSS style files eliminate the need to use many HTML commands, such as the FONT command that specifies the type of font, size, or color, as well as the TABLE command that is used to create tables, in other words anyone using CSS can design their entire site without using the tables as followed In traditional methods of page design, this does not mean that tables will not be used with CSS, as tables have another function which is to display and format data that need to be within a table. W3C has created this technology and made it a standard web design standard, and many sites have started to move from the traditional design method using tables to the new method using CSS technology. What do we use CSS flowchart style pages? First: Because it is currently an agreed upon standard, and the standards make things easier and easier, and also because CSS technology will be supported in the near future by all companies working in the field of information technology, which leads to their widespread use and therefore must Web page designers and developers learn it from now so that it is ready for the future. Second: Because the CSS flowchart style pages will be compatible with future hardware and software, and therefore they will be used in many areas that may appear in the future and we cannot predict now. Third: separating the content from the design, and this leads to simplification of the site management process, because the design can be controlled from one file, so some elements in the site design can be added and modified without the need to modify all the site files. Fourth: Making the site files smaller, which leads to lower operating costs. It also helps your site visitor, regardless of the speed of his internet connection, to summon the site’s pages through his browser of any kind in a way faster than before.
